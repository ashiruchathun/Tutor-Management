Relationship Explanation

1. Tutor inherits from Person.
   Tutor gets the common person fields: name, email, and phone.

2. TutorController uses TutorService.
   The controller receives API requests from the frontend and passes work to the service layer.

3. TutorServiceImpl implements TutorService.
   TutorService defines the methods, and TutorServiceImpl contains the actual business logic.

4. TutorServiceImpl uses TutorRepository.
   The service calls the repository to save, update, delete, and search tutor records.

5. TutorRepository extends JpaRepository.
   Spring Data JPA provides database functions such as save, findAll, findById, and delete.

6. TutorRepository manages Tutor entities.
   Tutor data is stored in the MySQL tutors table.

7. ReactPageController serves React frontend routes.
   It forwards routes like /tutors and /admin/tutors to index.html.

8. SecurityConfig controls login and route access.
   It protects admin pages and write API operations.
